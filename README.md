## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Usage

1. **Setup Page**: Enter up to 8 habits you want to track in 2026
2. **Dashboard**: Click on any month to start tracking
3. **Month Tracker**: 
   - Check/uncheck boxes to mark habit completion
   - Add highlights in the left sidebar
   - All progress saves automatically!
